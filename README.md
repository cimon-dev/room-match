# RoommateMatching — Next.js demo scaffold

This repository contains a minimal Next.js scaffold converted from a single `index.html` demo. It splits the demo into simple pages and components for easier iteration.

Quick start (Windows):

```bash
npm install
npm run dev
```

Pages created:
- `/` — Landing (Hero)
- `/discover` — Profiles grid + filters sidebar
- `/messages` — Chat placeholder

Components:
- `NavBar`, `Hero`, `FiltersSidebar`, `ProfileCard`, `ProfileModal`

The original `index.html` remains in the workspace as the full demo reference.
